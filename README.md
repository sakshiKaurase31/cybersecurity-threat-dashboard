# 🔐 Cybersecurity Threat Intelligence Dashboard

## 📊 Project Overview
This project is a Power BI dashboard that analyzes network traffic data to detect cyber threats and attack patterns.

## 📁 Dataset
- Total Records: 10,000
- Attributes:
  - Source IP
  - Destination IP
  - Protocol (TCP, UDP, ICMP)
  - Bytes Sent / Received
  - Attack Type   

## 📌 Key Features
- Protocol Distribution Analysis
- Attack Type Identification
- Top Attacker IP Detection
- Traffic Analysis (Internal vs External)
- Attack Trend Over Time

## 📈 Key Insights
- Attack Rate: 4%
- TCP protocol dominates traffic (~77%)
- Most common attacks: Brute-force, Port Scan
- Majority traffic is external (~85%)

## 🛠 Tools Used
- Power BI
- Power Query
